# Config-Beispiele

Dieser Ordner enthaelt **redaktierte Beispiel-Outputs**, die zeigen, was die Skripte
waehrend eines Runs schreiben.

Hinweise:
- Echte Ausfuehrungs-Logs duerfen nicht committed werden (sie enthalten oft Maschinen-/Benutzerdaten).
- Runtime-Logs sollen per Default ausserhalb des Repos geschrieben werden (siehe `scripts/Invoke-WinSetup.ps1`).
